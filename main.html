Algorithm 01 — Classification

LogisticRegression

Supervised O(n · d) Beginner

Despite its name, Logistic Regression is a classification algorithm. It models the probability that an input belongs to a class using the sigmoid function — elegant, interpretable, and remarkably powerful for linearly separable data.

Logistic Regression estimates the probability P(y=1 | x) by passing a linear combination of features through the sigmoid function, which squashes any real number into [0, 1].

σ(z) = 1 / (1 + e⁻ᶻ)   where   z = wᵀx + b
Loss = −[y log(ŷ) + (1−y) log(1−ŷ)]

Key Insight: The decision boundary is linear in feature space. If your classes are not linearly separable, consider adding polynomial features or switching to a tree-based method.

Advantages

  • Highly interpretable coefficients
  • Fast training and inference
  • Outputs calibrated probabilities
  • Works well with small datasets
  • Regularisation via L1/L2

Limitations

  • Assumes linear decision boundary
  • Sensitive to outliers
  • Requires feature engineering for non-linear patterns
  • Struggles with class imbalance
Best Used For
Spam Detection Credit Scoring Disease Prediction Sentiment Analysis A/B Testing
How It Works — Step by Step
1

Compute Linear Score

Calculate z = w₁x₁ + w₂x₂ + … + b for each sample using current weights.

2

Apply Sigmoid

Transform z into a probability ŷ = σ(z) ∈ (0,1). Values above 0.5 → class 1.

3

Compute Loss

Binary cross-entropy measures the divergence between ŷ and true label y.

4

Gradient Descent

Update w ← w − α · ∂L/∂w. Repeat until convergence.

Python — From Scratch (NumPy only)
import numpy as np

class LogisticRegression:
    """
    Logistic Regression implemented from scratch using NumPy.
    Uses gradient descent with binary cross-entropy loss.
    """
    def __init__(self, learning_rate=0.01, n_iterations=1000, 
                 regularisation=None, lambda_=0.01):
        self.lr = learning_rate
        self.n_iter = n_iterations
        self.reg = regularisation   # 'l1' or 'l2' or None
        self.lambda_ = lambda_
        self.weights = None
        self.bias = None
        self.losses = []

    @staticmethod
    def _sigmoid(z):
        # Clip for numerical stability
        z = np.clip(z, -500, 500)
        return 1.0 / (1.0 + np.exp(-z))

    def _compute_loss(self, y, y_hat):
        n = len(y)
        eps = 1e-15  # prevent log(0)
        y_hat = np.clip(y_hat, eps, 1 - eps)
        bce = -np.mean(y * np.log(y_hat) + (1 - y) * np.log(1 - y_hat))

        # Add regularisation term
        if self.reg == 'l2':
            bce += (self.lambda_ / (2 * n)) * np.sum(self.weights ** 2)
        elif self.reg == 'l1':
            bce += (self.lambda_ / n) * np.sum(np.abs(self.weights))
        return bce

    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0.0

        for i in range(self.n_iter):
            # Forward pass
            z = X @ self.weights + self.bias
            y_hat = self._sigmoid(z)

            # Compute loss
            self.losses.append(self._compute_loss(y, y_hat))

            # Backward pass
            error = y_hat - y
            dw = (1 / n_samples) * X.T @ error
            db = (1 / n_samples) * np.sum(error)

            # Regularisation gradient
            if self.reg == 'l2':
                dw += (self.lambda_ / n_samples) * self.weights
            elif self.reg == 'l1':
                dw += (self.lambda_ / n_samples) * np.sign(self.weights)

            # Gradient descent update
            self.weights -= self.lr * dw
            self.bias    -= self.lr * db

        return self

    def predict_proba(self, X):
        z = X @ self.weights + self.bias
        return self._sigmoid(z)

    def predict(self, X, threshold=0.5):
        return (self.predict_proba(X) >= threshold).astype(int)

    def score(self, X, y):
        return np.mean(self.predict(X) == y)

Algorithm 02 — Classification / Regression

DecisionTree

Supervised O(n·d·log n) Beginner

Decision Trees recursively split the feature space using the most informative feature at each step, building an interpretable, tree-shaped rule system. The foundation of ensemble methods like Random Forest and XGBoost.

At each node, the algorithm searches for the feature f and threshold t that maximise information gain — the reduction in impurity after the split.

Gini(S) = 1 − Σ pᵢ²
InfoGain = Impurity(parent) − Σ (|child|/|parent|)·Impurity(child)

Overfitting Warning: Unpruned trees memorise training data. Control via max_depth, min_samples_split, or min_samples_leaf.

Advantages

  • Completely interpretable & visualisable
  • Handles mixed feature types
  • No feature scaling needed
  • Captures non-linear boundaries

Limitations

  • High variance — prone to overfitting
  • Unstable to small data changes
  • Biased toward features with more levels
Best Used For
Medical Diagnosis Rules Extraction Feature Selection Explainable AI
1

Find Best Split

For every feature, try all thresholds. Choose (feature, threshold) maximising information gain.

2

Split the Node

Partition samples into left (x ≤ t) and right (x > t) subsets.

3

Recurse

Repeat on each child node until stopping criteria are met.

4

Assign Leaf Labels

Majority class (classification) or mean value (regression) at each leaf.

Python — Decision Tree From Scratch
import numpy as np
from collections import Counter

class Node:
    def __init__(self, feature=None, threshold=None, left=None, 
                 right=None, value=None):
        self.feature   = feature      # Feature index for split
        self.threshold = threshold    # Split threshold
        self.left      = left         # Left subtree
        self.right     = right        # Right subtree
        self.value     = value        # Leaf node prediction

    def is_leaf(self):
        return self.value is not None


class DecisionTree:
    def __init__(self, max_depth=10, min_samples_split=2, criterion='gini'):
        self.max_depth = max_depth
        self.min_samples = min_samples_split
        self.criterion = criterion
        self.root = None

    def _impurity(self, y):
        if len(y) == 0:
            return 0
        if self.criterion == 'gini':
            counts = np.bincount(y)
            probs = counts / len(y)
            return 1 - np.sum(probs ** 2)
        elif self.criterion == 'entropy':
            counts = np.bincount(y)
            probs = counts[counts > 0] / len(y)
            return -np.sum(probs * np.log2(probs))

    def _information_gain(self, y, left_y, right_y):
        n = len(y)
        weighted = (len(left_y)/n)*self._impurity(left_y) + \
                   (len(right_y)/n)*self._impurity(right_y)
        return self._impurity(y) - weighted

    def _best_split(self, X, y):
        best_gain, best_feature, best_threshold = -np.inf, None, None
        n_features = X.shape[1]

        for f in range(n_features):
            thresholds = np.unique(X[:, f])
            for t in thresholds:
                left_mask  = X[:, f] <= t
                right_mask = ~left_mask
                if left_mask.sum() == 0 or right_mask.sum() == 0:
                    continue
                gain = self._information_gain(y, y[left_mask], y[right_mask])
                if gain > best_gain:
                    best_gain, best_feature, best_threshold = gain, f, t

        return best_feature, best_threshold

    def _build(self, X, y, depth=0):
        # Stopping conditions
        if (depth >= self.max_depth or 
            len(y) < self.min_samples or 
            len(np.unique(y)) == 1):
            return Node(value=Counter(y).most_common(1)[0][0])

        feature, threshold = self._best_split(X, y)
        if feature is None:
            return Node(value=Counter(y).most_common(1)[0][0])

        mask = X[:, feature] <= threshold
        left  = self._build(X[mask], y[mask], depth + 1)
        right = self._build(X[~mask], y[~mask], depth + 1)
        return Node(feature=feature, threshold=threshold, 
                   left=left, right=right)

    def fit(self, X, y):
        self.root = self._build(X, y)
        return self

    def _traverse(self, x, node):
        if node.is_leaf():
            return node.value
        if x[node.feature] <= node.threshold:
            return self._traverse(x, node.left)
        return self._traverse(x, node.right)

    def predict(self, X):
        return np.array([self._traverse(x, self.root) for x in X])

    def score(self, X, y):
        return np.mean(self.predict(X) == y)

Algorithm 03 — Clustering

K-MeansClustering

Unsupervised O(n·k·d·i) Beginner

K-Means partitions data into k clusters by iteratively assigning points to their nearest centroid and recomputing centroids. Deceptively simple, yet forms the backbone of customer segmentation and image compression.

K-Means minimises the within-cluster sum of squares (WCSS) — the total squared distance from each point to its cluster centroid.

WCSS = Σₖ Σᵢ∈Cₖ ‖xᵢ − μₖ‖²

Choosing k: Use the Elbow Method — plot WCSS vs k and look for the point of diminishing returns. The Silhouette Score provides a more rigorous measure of cluster quality.

Advantages

  • Simple and fast — O(n·k·d·i)
  • Scales to large datasets
  • Easily interpretable clusters

Limitations

  • Must specify k in advance
  • Sensitive to initialisation
  • Assumes spherical, equal-sized clusters
  • Fails on non-convex shapes
Best Used For
Customer Segmentation Image Quantisation Document Clustering Anomaly Detection
1

Initialise Centroids

Choose k random points (or use K-Means++ for smarter init).

2

Assign Clusters

Each point joins the cluster of its nearest centroid (Euclidean distance).

3

Update Centroids

New centroid = mean of all points in the cluster.

4

Repeat

Iterate steps 2–3 until assignments stop changing or max_iter reached.

Python — K-Means From Scratch (with K-Means++)
import numpy as np

class KMeans:
    """K-Means with K-Means++ initialisation."""
    def __init__(self, k=3, max_iter=300, tol=1e-4, n_init=10):
        self.k = k
        self.max_iter = max_iter
        self.tol = tol
        self.n_init = n_init  # Run multiple times, keep best
        self.centroids_ = None
        self.labels_ = None
        self.inertia_ = None

    def _init_centroids_plusplus(self, X):
        """K-Means++ smart initialisation."""
        n = X.shape[0]
        idx = np.random.randint(n)
        centroids = [X[idx]]

        for _ in range(self.k - 1):
            dists = np.array([
                min(np.sum((x - c)**2) for c in centroids) for x in X
            ])
            probs = dists / dists.sum()
            cumprobs = np.cumsum(probs)
            r = np.random.rand()
            idx = np.searchsorted(cumprobs, r)
            centroids.append(X[idx])

        return np.array(centroids)

    def _assign(self, X, centroids):
        dists = np.array([
            np.sum((X - c)**2, axis=1) for c in centroids
        ])  # shape: (k, n)
        return np.argmin(dists, axis=0)

    def _compute_inertia(self, X, labels, centroids):
        return sum(
            np.sum((X[labels == k] - centroids[k])**2)
            for k in range(self.k)
        )

    def _fit_once(self, X):
        centroids = self._init_centroids_plusplus(X)

        for _ in range(self.max_iter):
            labels = self._assign(X, centroids)
            new_centroids = np.array([
                X[labels == k].mean(axis=0) if (labels == k).any()
                else centroids[k]
                for k in range(self.k)
            ])
            if np.linalg.norm(new_centroids - centroids) < self.tol:
                break
            centroids = new_centroids

        return labels, centroids, self._compute_inertia(X, labels, centroids)

    def fit(self, X):
        best_inertia = np.inf
        for _ in range(self.n_init):
            labels, centroids, inertia = self._fit_once(X)
            if inertia < best_inertia:
                best_inertia = inertia
                self.centroids_ = centroids
                self.labels_ = labels
                self.inertia_ = inertia
        return self

    def predict(self, X):
        return self._assign(X, self.centroids_)

    def elbow_plot(self, X, k_range=range(1, 11)):
        """Returns WCSS for each k — plot to find elbow."""
        inertias = []
        for k in k_range:
            m = KMeans(k=k, n_init=3)
            m.fit(X)
            inertias.append(m.inertia_)
        return list(k_range), inertias

PyTorch — GPU-Accelerated K-Means
import torch
import torch.nn.functional as F
from sklearn.datasets import make_blobs

device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')

class KMeansTorch:
    """
    Vectorised K-Means with GPU support via PyTorch.
    Significantly faster for large n with CUDA.
    """
    def __init__(self, k, max_iter=300, tol=1e-4):
        self.k = k
        self.max_iter = max_iter
        self.tol = tol
        self.centroids = None

    def _pairwise_sq_dist(self, A, B):
        # A: (n, d), B: (k, d) → (n, k)
        return (torch.cdist(A, B, p=2) ** 2)

    def fit(self, X: torch.Tensor):
        n, d = X.shape
        # K-Means++ init
        idx = torch.randint(n, (1,)).item()
        centroids = [X[idx]]
        for _ in range(self.k - 1):
            dists = torch.stack([
                torch.sum((X - c)**2, dim=1) for c in centroids
            ]).min(dim=0).values
            probs = dists / dists.sum()
            idx   = torch.multinomial(probs, 1).item()
            centroids.append(X[idx])

        self.centroids = torch.stack(centroids)

        for _ in range(self.max_iter):
            # Assign
            dists  = self._pairwise_sq_dist(X, self.centroids)  # (n, k)
            labels = torch.argmin(dists, dim=1)                  # (n,)

            # Update
            new_centroids = torch.stack([
                X[labels == k].mean(dim=0) if (labels == k).any()
                else self.centroids[k]
                for k in range(self.k)
            ])

            shift = torch.norm(new_centroids - self.centroids)
            self.centroids = new_centroids
            if shift < self.tol:
                break

        self.labels_ = labels
        return self

    def predict(self, X: torch.Tensor):
        dists = self._pairwise_sq_dist(X, self.centroids)
        return torch.argmin(dists, dim=1)


# ── Usage ──────────────────────────────────────
X_np, _ = make_blobs(n_samples=10000, centers=5, random_state=42)
X = torch.FloatTensor(X_np).to(device)

model = KMeansTorch(k=5)
model.fit(X)

print(f"Cluster sizes: {torch.bincount(model.labels_).tolist()}")
Algorithm 04 — Dimensionality Reduction

Principal ComponentAnalysis

Unsupervised O(n·d²) Intermediate

PCA finds the directions of maximum variance in high-dimensional data and projects it onto a lower-dimensional subspace. Essential for visualisation, noise reduction, and decorrelating features before applying other algorithms.

PCA decomposes the covariance matrix of centred data to find eigenvectors (principal components) ranked by their eigenvalues (explained variance).

Cov(X) = (1/n) · XᵀX
Cov(X) · v = λ · v   (eigen-decomposition)
X_reduced = X · V[:, :k]

How many components? Choose k such that the cumulative explained variance ratio exceeds 90–95%. Use a scree plot to visualise the trade-off.

Best Used For
2D/3D Visualisation Noise Reduction Feature Compression Preprocessing Face Recognition
1

Centre the Data

Subtract the mean of each feature: X = X − μ

2

Compute Covariance Matrix

Σ = (1/n) · XᵀX — captures how features vary together.

3

Eigen-decomposition

Find eigenvectors V and eigenvalues λ. Sort by λ descending.

4

Project

Select top-k eigenvectors. X_reduced = X · V[:, :k]

PCA — Principal Components on 2D Data
// Principal Component Analysis ALGORITHM PCA(X, n_components): // Step 1: Centre the data μ ← mean(X, axis=0) // [d] feature means X_centred ← X − μ // Step 2: Covariance matrix Σ ← (1/n) · X_centredᵀ · X_centred // [d × d] // Step 3: Eigen-decomposition eigenvalues, eigenvectors ← eig(Σ) // Step 4: Sort by eigenvalue (descending) idx ← argsort(eigenvalues)[::-1] eigenvalues ← eigenvalues[idx] eigenvectors ← eigenvectors[:, idx] // Step 5: Select top-k components V ← eigenvectors[:, :n_components] // [d × k] // Step 6: Project data X_reduced ← X_centred · V // [n × k] // Explained variance ratio evr ← eigenvalues[:k] / sum(eigenvalues) RETURN X_reduced, V, evr // Alternative: SVD is numerically more stable U, S, Vᵀ ← SVD(X_centred / √n) components ← Vᵀ[:n_components] // [k × d] X_reduced ← X_centred · componentsᵀ
Python — PCA From Scratch (SVD method)
import numpy as np

class PCA:
    """
    PCA via SVD decomposition — numerically stable.
    """
    def __init__(self, n_components=None, whiten=False):
        self.n_components = n_components
        self.whiten = whiten
        self.components_ = None         # Principal axes [k, d]
        self.explained_variance_ = None # Eigenvalues
        self.explained_variance_ratio_ = None
        self.mean_ = None

    def fit(self, X):
        n, d = X.shape
        self.n_components = self.n_components or min(n, d)
        self.mean_ = X.mean(axis=0)
        X_c = X - self.mean_

        # SVD of centred data
        U, S, Vt = np.linalg.svd(X_c, full_matrices=False)

        # Eigenvalues from singular values
        eigenvalues = (S ** 2) / (n - 1)
        total_var = eigenvalues.sum()

        self.components_ = Vt[:self.n_components]
        self.explained_variance_ = eigenvalues[:self.n_components]
        self.explained_variance_ratio_ = self.explained_variance_ / total_var
        self.singular_values_ = S[:self.n_components]
        return self

    def transform(self, X):
        X_c = X - self.mean_
        X_t = X_c @ self.components_.T  # [n, k]
        if self.whiten:
            X_t /= np.sqrt(self.explained_variance_)
        return X_t

    def fit_transform(self, X):
        return self.fit(X).transform(X)

    def inverse_transform(self, X_reduced):
        return X_reduced @ self.components_ + self.mean_

    def reconstruction_error(self, X):
        X_r = self.inverse_transform(self.transform(X))
        return np.mean((X - X_r) ** 2)

    def plot_scree(self):
        """Print scree plot data."""
        cumulative = np.cumsum(self.explained_variance_ratio_)
        for i, (ev, cev) in enumerate(zip(self.explained_variance_ratio_, cumulative)):
            bar = '█' * int(ev * 100)
            print(f"PC{i+1:2d}: {ev:.3f} ({cev:.3f} cum.) {bar}")
        return cumulative


# ── Usage ──────────────────────────────────────

print(f"Reduced shape:   {X_reduced.shape}")

pca.plot_scree()
print(f"\nReconstruction MSE: {pca.reconstruction_error(X):.4f}")
TensorFlow — Autoencoder (Non-linear PCA)
import tensorflow as tf
from tensorflow import keras
import numpy as np

class Autoencoder(keras.Model):
    """
    Autoencoder as non-linear generalisation of PCA.
    The bottleneck layer = compressed representation.
    """
    def __init__(self, input_dim, latent_dim, hidden_dims=[128, 64]):
        super().__init__()
        
        # Encoder
        encoder_layers = []
        prev_dim = input_dim
        for h in hidden_dims:
            encoder_layers += [
                keras.layers.Dense(h, activation='relu'),
                keras.layers.BatchNormalization(),
                keras.layers.Dropout(0.1)
            ]
        encoder_layers.append(keras.layers.Dense(latent_dim, name='bottleneck'))
        self.encoder = keras.Sequential(encoder_layers, name='encoder')
        
        # Decoder (mirror of encoder)
        decoder_layers = []
        for h in reversed(hidden_dims):
            decoder_layers += [
                keras.layers.Dense(h, activation='relu'),
                keras.layers.BatchNormalization(),
            ]
        decoder_layers.append(keras.layers.Dense(input_dim, activation='linear'))
        self.decoder = keras.Sequential(decoder_layers, name='decoder')

    def call(self, x, training=False):
        z = self.encoder(x, training=training)
        return self.decoder(z, training=training)

    def encode(self, x):
        return self.encoder(x, training=False)

from sklearn.datasets import load_digits
from sklearn.preprocessing import MinMaxScaler

    X, X, epochs=100, batch_size=64,
    validation_split=0.15,
